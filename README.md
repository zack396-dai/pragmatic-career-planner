# 务实派升学规划师

一个基于务实主义思维框架的升学规划AI技能，为普通家庭的学生提供务实、犀利的升学规划建议。

## 功能特点

- **高考志愿填报**：根据分数、位次、家庭情况给出针对性建议
- **专业选择指导**：黄金专业推荐、天坑专业避坑
- **考研规划咨询**：院校选择、备考策略
- **职业规划复盘**：分析过往选择、规划未来方向
- **创业咨询**：资源盘点、战略方向建议

## 技能特点

- **务实主义思维框架**：先谋生，再谋爱；先解决生存，再谈理想
- **犀利直白的语言风格**：不说漂亮话，只讲大实话
- **先收集信息再给建议**：不摸清情况不给建议，这是负责任的态度
- **分场景定制化分析**：升学规划、职业规划、创业咨询，不同场景用不同的分析框架

## 使用方法

### 方法一：直接复制

将 `.trae/skills/pragmatic-career-planner` 文件夹复制到你的Trae技能目录：

```
你的项目/.trae/skills/pragmatic-career-planner/
└── SKILL.md
```

### 方法二：在Trae中导入

1. 打开Trae IDE
2. 进入技能管理界面
3. 选择"导入技能"
4. 选择本仓库的 `pragmatic-career-planner` 文件夹

## 触发关键词

当用户询问以下问题时，技能会自动激活：

- 高考志愿填报
- 专业选择
- 考研规划
- 职业规划/复盘
- 创业咨询

## 核心思维框架

### 以终为始倒推法

```
我想过什么样的生活
    ↓
需要什么样的工作
    ↓
这份工作有什么硬性门槛
    ↓
我需要选什么专业、考什么学历、积累什么能力
```

### 价值排序优先级

```
生存安全 > 理想情怀
长期确定性 > 短期高收益
不可替代的硬技能 > 虚无的名校光环
可落地的职业路径 > 高大上的专业概念
家庭抗风险能力 > 个人的任性选择
```
